Questionários:

O projeto de sites de questionários foi desenvolvido com foco na funcionalidade e simplicidade, atendendo tanto a demandas de uso pessoal quanto às necessidades de instituições escolares que buscam uma ferramenta prática para coletar respostas

A estrutura HTML da página principal inclui um cabeçalho com o nome ou logotipo do site, seguido por um formulário de questionário Cada campo do formulário (perguntas, opções de múltipla escolha ou respostas abertas) é organizado de maneira clara dentro de divisões específicas (.questionario) Adicionalmente, há um rodapé com informações sobre como acessar resultados ou enviar as respostas

O design CSS define um estilo limpo e intuitivo, utilizando cores suaves que garantem conforto visual durante o preenchimento do questionário A tipografia foi escolhida para facilitar a leitura e organização do conteúdo Botões de envio e navegação possuem estilos chamativos, garantindo que os usuários encontrem facilmente as ações desejadas A página é completamente responsiva, permitindo que questionários sejam acessados e preenchidos em diferentes dispositivos, como smartphones, tablets e desktops

Divulgações:

O projeto de sites de divulgação foi desenvolvido como uma página web funcional e visualmente atraente, utilizando HTML e CSS O objetivo principal é permitir que clientes ou empresas promovam informações, produtos ou serviços de forma clara e organizada, garantindo uma experiência intuitiva para o público-alvo

A estrutura HTML organiza o site com elementos semânticos que facilitam a compreensão do conteúdo O cabeçalho apresenta o nome ou logo do projeto de divulgação, acompanhado por um menu de navegação A seção principal destaca as informações principais com um design responsivo que se adapta a diferentes dispositivos Divisões adicionais foram criadas para apresentar conteúdos específicos, como descrição dos serviços, depoimentos ou galerias de imagens

O design CSS prioriza a personalização para refletir a identidade visual do cliente Foi utilizada uma paleta de cores neutra e flexível, ajustável às preferências específicas A tipografia foi cuidadosamente escolhida para transmitir profissionalismo e clareza, enquanto efeitos de hover em botões e links melhoram a interatividade Todo o layout é responsivo, garantindo acessibilidade e uma boa experiência de navegação em dispositivos móveis e desktops
